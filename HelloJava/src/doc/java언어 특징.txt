자바란?

자바 언어 특징

#운영체제(Operating System, 플랫폼)에 독립적이다.
자바로 작성한 프로그램은 운영체제의 종류에 관계없이 실행이 가능하다

#객체지향 언어이다
객체지향개념의 특징인 상속, 캡슐화, 다형성 등이 잘 적용된 
순수한 객체지향언어

#자동 메모리 관리(Garbage Collection)
가비지컬렉터가 자동적으로 메모리를 관리해 준다
즉, 몇 가지를 제외하고 대부분 프로그래머가 메모리를 따로 
관리하지 않아도 된다

#네트워크와 분산처리를 지원한다

#멀티쓰레드를 지원한다

#동적 로딩(Dynamic Loading)을 지원한다
실행 시에 모든 클래스가 로딩되지 않고 필요한 시점에 
클래스를 로딩하여 사용할 수 있어 메모리를 효율적으로 사용 가능

#JVM(Java Virtual Machine)
자바 가상 머신, 자바를 실행하기 위한 가상 기계
이 것 덕분에 운영체제에 독립이 가능
